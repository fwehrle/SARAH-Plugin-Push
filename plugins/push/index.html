Le module push permet d'envoyer une notification push avec le service Prowl.

Il utilise la librairie node-prowl développée par Mathias Pettersson, mape@mape.me


Exemple: SARAH, demande à Franck de chercher du pain.

Ce plugin utilise les API Prowl pour envoyer des notifications push.
Pour connaitre potre clé API, allez sur le site de Prowl. Sur iPhone, il faut installer une application Prowl iOS.
